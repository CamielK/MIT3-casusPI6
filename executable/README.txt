To execute the prototype application:


1. execute the application by double clicking 'MIT3prototype.exe' on a windows machine.


	OR

(using JRE)
1. make sure you have a JRE or JDK installed with version 1.7.0_79 or above. 
	(Get JRE at http://www.oracle.com/technetwork/java/javase/downloads)

2. execute the application by double clicking 'Mit3Prototype.jar'
